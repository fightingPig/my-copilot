# Java 编码规范

请始终遵守以下要求：

1. 禁止使用 `Executors` 快捷工厂创建线程池，必须手动构造 `ThreadPoolExecutor`。
2. 禁止使用 `System.out.println` / `System.err.println` 打印业务日志。
3. 禁止硬编码密码、Token、AK/SK、密钥。
4. 禁止捕获通用 `Exception` / `Throwable`。
5. 禁止 DAO 层编写业务逻辑。
6. 禁止 `select *`、全表更新、全表删除。
7. 修复代码时只改违规点，不改动无关业务逻辑。

更完整规则见 `.claude/java-coding-standards.md`。
